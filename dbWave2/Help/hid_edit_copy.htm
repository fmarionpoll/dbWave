Copy command (Edit menu)

Use this command to copy selected data onto the clipboard. This command is currently (Dec 2013) implemented under 4 view modes:

  • Data view
  • Spike detection
  • Spike view
  • Time series

Copying data to the clipboard replaces the contents previously stored there.

The data transferred to the clipboard are the curves currently displayed on the screen. You can then paste these curves under CorelDraw (preferred) or Word / Powerpoint.
