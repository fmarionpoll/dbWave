Create a database.

To create your first database, either click on the "File new" icon or go to the menu File / New... and select database. This will create a database which should be stored on your computer as an "*.mdb" file. As a rule, dbWave will use your connection name as the base of the database name and append a number which will be incremented when you create further databases.

This database will be initially empty. dbWave works by maintaining a database that collects information stored into primary data files (*.dat). You will fill the database by acquiring data with your data acquisition card, or by importing data acquired with another program if dbWave can read their format.

The screen is organized in 3 panels. The central panel (or "View") displays the data you have acquired and on which you work on. The upper panel display the properties of the file currently selected. The right panel deals with the database and allows you to select a subset of your records. On the top of these 3 panels, you find a row of icons with text. These icons display which operations are currently available. Since you have no data file in the database, the only option available is "Record". On top of this row of icons, you find another row of buttons which are used in the different sections of the program.
