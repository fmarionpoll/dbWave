Start from scratch

You have just installed dbWave and you have launched it. Now what?

  1. Create a database
  2. Acquire data
  3. Define amplification settings
  4. Define sampling settings
  5. Define where data will be stored
  6. Start recording
  7. Et voila!

Important:

When you exit dbWave, sometimes the database is not saved and the settings you have used during your session are not saved. Go to "My Documents" and make sure that a directory called "My dbWave Files" is present. If not, create this folder.
