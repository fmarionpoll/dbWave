Spike files

Spike files are files generated by dbWave when spikes are detected. These files have a "*.spk" extension. They consist into a collection of snippets from the original datafile from which spikes were detected.

Each "spike" includes an array with the spike data waveform (typically 60 points), and additional descriptors such as the time of its occurence, and it's class number.

Spike files are usually stored along with data files. The name of each spike file is generated by using the name of the data file and adding ".spk".

One spike file can contain spike trains from different neurons or channels, as well as stimulus on/off descriptors.
