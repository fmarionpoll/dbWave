Tools / Export data files as text...

This routine will read all data files currently defined in the database and convert them into ASCII text files.

The file contains a header (line 1 to 19) and then the data in mv, each row holding the value read by the card at consecutive intervals. In this example, data were acquired on 2 channels during 6 seconds at 10 kHz:

--------------------------------------

  1. filename=c:\data\alice\chr2-gr66a\13-11-14\folder1\gr66a-chr2000002.dat
  2. acqdate= 13 November 2013
  3. acqtime= 17:14:19
  4. sampling rate(Hz)= 10000.000000
  5. nsamples=20000
  6. nchans=2
  7. comment1=caffbl
  8. stim1=BL
  9. conc1=#on
  10. stim2=Quinine
  11. com2=#0
  12. insect=drosophila
  13. location=Proboscis
  14. sensillum=L5
  15. strain=Gr66a(M)xChR2(F)
  16. sex=female
  17. operator=alice
  18. comment2=
  19. tip, channel 2
