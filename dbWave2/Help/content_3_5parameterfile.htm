Preferences file (*.prefs)

A preferences file (*.prefs) file is stored in "My dbWave Files" directory. This file contains the user last selection of a number of different parameters like display and printing options, or data acquisition options. These parameters are stored under a binary format.

Under normal circumstances, you do not need to care about this file. However, you may need to erase this file if your program can't start again after you have selected options that made it crash when the program starts again.
