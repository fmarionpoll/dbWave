Define where data will be stored

Once you have defined the amplifier settings and the sampling parameters, you can hit the "Start" button (see screen shot here). This will launch the following dialog box.

This dialog box defines where your data will be stored and how your data will be described in the database.

Filename and path, define where data are stored. Data file names are composed of a root (for example "data"), to which is appended a number. This number will be automatically incremented each time this dialog box is called. If you need to acquire data in a quick succession, click "acquire a series of files" so that the dialog box is bypassed.

The other settings of this dialog box represent fields used in the database to describe your data. Currently, the list of fields is fixed. This information will be saved within the data file, into a header.

 

Clicking "OK" will start the acquisition.
