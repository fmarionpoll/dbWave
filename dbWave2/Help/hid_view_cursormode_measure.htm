Cursor mode: Measure

This changes the cursor which has different functions depending on the view mode:

  • under data display mode, this cursor allows you to set cursors on the view and to perform measurements, either of amplitude between two horizontal cursors or at vertical cursors placed along the curve at different time intervals. Whether it is in horizontal or vertical measuring mode, is determined by parameters adjusted when you hit the yellow button on the left of the toolbar.
  •  under spike detection mode, it allows you to define a stimulus, by placing vertical cursors on the curves which will be interpreted as a stimulus signal going on and off. In order to add a stimulus event, left click on the mouse where you want to add it. You can change this later, either by moving the vertical cursor (in arrow mode, by clicking on it and keeping it clicked, the cursor will move with our mouse cursor), or by editing the stimulus series (in the menu "Tools" / "Edit stimulus...").
  •  under spike edit mode, it allows you to either add new spikes manually, or to change the spike class which is located at the vertical of your cursor when you left-click the mouse. If you click in an area where there are no spikes, it will create a new spike. If you click over a spike, it will select that spike and change its class (according to the parameters you have defined on that view). The program allows for some jitter in the placement of the cursor, so that if you click the mouse in the vicinity of a spike, it will select that spike and change its class . If you want to force the creation of a spike, press "Ctrl" while you depress the left button of the mouse.
  •  this command (button) has no function under the other views (database, spike sorting, spike templates, spike time series analysis, data acquisition).
