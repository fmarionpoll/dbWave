Export Data Dialog

This dialog is called by the menu item "File / Export data file..." which is available under the view "Data" (see here).

Source file: name of file to be transformed/exported. It can be either a single file, or the whole list. This dialog box just display the names of the files without the path.

Export as: format under which the data files will be saved. They will be saved into the same directory as the source file.

  • dbWave is the genuine dbWave format. It might be useful if one wants to extract a channel or to undersample the data. Each exported file will have the same name as the original file, except that the prefix "AW_" will be added to their name.
  • Sapid 3.5 is the format read by a specific program (written by Bev Mitchell) which had been designed to sort spikes from extracellular recordings from insect sensilla. Each file name will be concatenated with the prefix "SAP_".
  • ASCII text is a simple text file, where data are saved; each row is one sampling interval while each column is from a different channel. Each file will be prefixed with "ASC_" and its extension will be ".TXT" (instead of ".dat").
  • Excel is a spreadsheet that you can read with Microsoft Excel. The file will have the extension ".xls". As Excel files had a limitation in the number of rows, data which have more than 16384 rows are spread across consecutive columns.

Save comments in a separate file: tick this option if you want to store descriptors of each data file into a different file.

Export options:

  • all channels: export all data channels.

  • single channel: export only one data channel. If selected, this option will allow the user to select a given channel. This is useful if the data recorded include additional channels which are not used for the analysis.

  • complete file or from (s) t1 to t2: define here if you want to export the entire recording or just a part of it.

  • undersample ratio: 1 to n - you can export either the raw data, or you can undersample the data by exporting only 1 point under 2 or 10 (ad libitum). Each point exported will be then computed as the average of 2 or 10 consecutive points.

Export:: transform a single file. A message box will be displayed upon completion of the operation and the name of the file which has been exported is removed from the dropdown list.

Export All: transform all files of the list.
