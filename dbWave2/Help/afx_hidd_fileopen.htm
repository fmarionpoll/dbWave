File Open dialog box

The following options allow you to specify which file to open:

File Name

Specifies the file you want to open. This box lists files with the extension you select in the List Files of Type field. You can select a single file or a list of file (press SHIFT while selecting with the mouse or the keyboard)

List Files of Type

Specifies the type of file you want to open:

  • Database files (*.mdb). Database files are files storing information about a series of electrophysiological recordings. Each data acquisition file is storing descriptors along with the recorded data, and these descriptors are read by dbWave and stored into a database file. This file can be read by earlier versions of Microsoft Access (up to 2007).
  • Data acquisition files (*.dat). You can also open data files from other programs with the extension *.asd or *.mcid. Opening such files will open a database file in which information read into these files will be stored.
  • Spike files (*.spk). Files generated by dbWave after detecting and sorting spikes. Opening such files will open a database file in which information read into these files will be stored.
  • Project files (*.prj). Project files contain a list of file names which can be subsequently opened by dbWave.
  • Text files (*.txt). Text files contain only text.

Drives

Specifies the drive in which dbWave stores the file that you want to open.

Directories

Specifies the directory in which dbWave stores the file that you want to open.

Network

Click this button to connect to a network location, assigning it a new drive letter.
